=== SIG Main Documentation ===

Overall description, tutorials, and specification
of internal file formats is available as a tex 
project in the sigdoc folder. It includes project
files for TeXnic Center to be used in Windows.

The produced document is sigdoc/sigdoc.pdf.

== REFERENCE MANUAL ==

Doxygen is used to generate a reference manual:

1) download and install doxygen in your system:
http://www.stack.nl/~dimitri/doxygen/download.html

2) from the command prompt, run:
doxygen doxyfile.txt
The html entry point will be refman/html/index.html

3) go to folder doxygen/latex/, and run make.bat
pdf reference manual be refman.pdf
